# Faulty Calculator

## 📌 About
This is a mini JavaScript project inspired by CodeWithHarry's Sigma Web Development Course.

## 🚀 Features
- Takes user input using prompt()
- Performs normal calculations
- 10% chance of giving incorrect results
- Uses Math.random() for randomness

## 🛠️ Technologies Used
- HTML
- JavaScript
